📱安装到手机桌面

1
用 Safari 打开此页面
必须使用 Safari 浏览器打开（其他浏览器无法添加到主屏幕）
2
点击底部「分享」按钮
找到 ↑ 分享图标，通常在底部工具栏中间
3
选择「添加到主屏幕」
在弹出的菜单中找到并点击「添加到主屏幕」
4
点击「添加」完成安装
回到桌面即可看到 Friend App 图标，点击即可使用
打开 Friend App（iOS）
1
用 Chrome 打开此页面
使用 Chrome 浏览器访问此链接
2
点击「添加到主屏幕」
浏览器会自动弹出安装提示，或点击菜单→「安装应用」
3
确认安装
点击「安装」，应用图标将出现在桌面
或

扫码在手机上打开此页面
